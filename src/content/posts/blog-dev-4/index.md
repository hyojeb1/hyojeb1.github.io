## 📐 WebGL: 삼각형 출력과 리소스 관리의 미학

브라우저라는 새로운 런타임에서 그래픽스 프로그래머의 인사를 건네보았습니다. WebGL을 이용해 가장 순수한 형태의 폴리곤인 **삼각형**&#8203;을 띄우는 데 성공했습니다.

```javascript title="src/scripts/webgl-core.js"
// WebGL Context 초기화 및 삼각형 렌더링
const gl = canvas.getContext('webgl');
if (!gl) {
    throw new Error('WebGL context 확보 실패. 브라우저 환경을 확인하세요.');
}
// 0과 1의 세계에서 가장 우아한 도형을 그립니다.
gl.drawArrays(gl.TRIANGLES, 0, 3);

```

하지만 여기까지입니다. WebGL의 세계를 더 깊게 탐험하는 것은 잠시 멈추기로 했습니다. 현재 저에게 가장 중요한 리소스는 **'시간'**&#8203;​이며, 가장 높은 수익률을 기대할 수 있는 곳은 곧 시작될 **졸업 프로젝트**​이기 때문입니다.

> [!info] 판단
> WebGL은 매력적이지만, 지금은 지난 4쿼터 **오로라 엔진**​의 과오를 복기하고 졸업 프로젝트를 위한 설계를 다지는 것이 전략적으로 우위에 있다고 판단했습니다.

## 📜 오로라 엔진 회고 시리즈를 예고하며
